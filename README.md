## Overview

This is the frontend for [FinSight](https://github.com/Rajveer-code/Finsight) — an end-to-end ML system that extracts alpha signals from 14,584 S&P 500 earnings call transcripts using FinBERT, RAG, and gradient-boosted models.

The dashboard visualises all research results interactively with animations, keyboard navigation, and mobile support.

---

## Pages

| Page | Description |
|---|---|
| **Overview** | Hero stats, pipeline diagram, sentiment trends, key findings |
| **Model Performance** | Walk-forward IC/AUC comparison, model toggle, stability analysis |
| **Feature Importance** | SHAP top-20 bar chart, group pie, insight cards |
| **Backtest** | 5-day vs 20-day equity curves, quarterly returns, PEAD analysis |
| **Sector Analysis** | IC by GICS sector, +0.311 Energy finding, sector insights |
| **Explorer** | Search any of 554 tickers, radar chart, sentiment history, scatter |

## Features

- **Animated stat counters** — count up from 0 on data load with skeleton states
- **Framer Motion page transitions** — smooth fade+slide between routes
- **Linear-style sidebar** — animated active pill using layoutId
- **Keyboard shortcuts** — press `?` to see navigation shortcuts (G+O, G+M, etc.)
- **Mobile responsive** — hamburger drawer on small screens
- **Scroll to top** button appears after 400px scroll
- **Console easter egg** — open DevTools to see it
- **Chart entrance animations** — every chart draws itself on first render
- **whileInView** — sections animate as you scroll down

## Data

All data is served as static JSON from `public/data/` — generated by the Python ML pipeline:

```
public/data/
├── stats.json           — summary statistics
├── model_results.csv    — walk-forward IC/AUC per model per year
├── shap.json            — SHAP feature importance values
├── backtest_5d.json     — 5-day holding period quarterly returns
├── backtest_20d.json    — 20-day holding period quarterly returns
├── sector_ic.json       — IC by GICS sector
├── explorer.json        — per-ticker per-quarter sentiment features
├── tickers.json         — ticker list with summary stats
├── year_dist.json       — transcript count by year
└── sentiment_by_year.json — mean sentiment trends
```

To regenerate from the ML pipeline:
```bash
cd ../finsight
python export_data.py
```
